## 1D

Performing the experiments:
Source the file experiments_emo21_1d_final.R
You need the file function_experiments_1d.R in the working directory of the used Cluster to perform the experiments.

Evaluating the experiments:
Source the file evaluate_experiments_emo21_1d_final.R
You need the files functions_evaluation_1d.R, result_1d_emo21_final.RData and true2d_emo21_final.RData in the used working directory.
